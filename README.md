# 在线教育管理系统

## 技术栈
Spring Boot、Spring Security、JWT、MyBatis、MySQL、PageHelper、Vue3、Element Plus

## 项目介绍
本项目是一个前后端分离的在线教育管理系统，支持管理员、教师、学生三类角色，主要包含用户认证、课程管理、学生选课、课程资源、在线考试、成绩统计、讨论互动等功能。

## 核心功能
- 用户注册、登录、JWT认证
- 基于角色的权限控制
- 课程发布与学生选课
- 在线考试与自动评分
- 成绩统计与学习进度展示
- 讨论互动与评论管理

## 运行环境
JDK 21
MySQL 8
Maven
Node.js 18+

## 项目说明
本项目主要用于 Java 后端开发学习与实践，重点实现了用户认证、权限控制、考试评分、成绩统计和多表分页查询等后端业务。
